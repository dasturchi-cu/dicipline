# REJABON AI — Foydalanuvchi Sayohatlari

## Sayohat 1: Birinchi ochish (Onboarding)

1. Ilova ochiladi → REJABON AI logotipi
2. 3 slayd: Hayotingiz markazi / Offline xavfsizlik / AI murabbiy
3. "Boshlash" tugmasi → Boshqaruv paneli
4. Bo'sh holat: "Birinchi vazifangizni qo'shing"
5. Tez harakat orqali vazifa yaratadi
6. Muvaffaqiyat animatsiyasi

**Natija:** Foydalanuvchi birinchi vazifasini yaratdi, ilova qiymatini tushundi.

---

## Sayohat 2: Kunlik rutina (Dilnoza)

1. Ertalab bildirishnoma: "Bugungi 5 ta vazifangiz bor"
2. Boshqaruv panelini ochadi → kunlik xulosa
3. Odatlar bo'limida bugungi odatlarni belgilaydi
4. Ta'lim modulida 2 soat o'qish sessiyasi qo'shadi
5. Kechqurun kundalik yozuv + kayfiyat
6. AI murabbiy: "Ajoyib kun! Streak 7 kun"

**Natija:** To'liq kunlik tsikl bajarildi.

---

## Sayohat 3: Moliya nazorati (Madina)

1. Moliya bo'limiga kiradi
2. "Daromad qo'shish" → summa, kategoriya
3. "Xarajat qo'shish" → do'kon xarajati
4. Statistika grafikini ko'radi
5. Oy oxirida balans xulosasi

**Natija:** Moliyaviy holat aniq ko'rinadi.

---

## Sayohat 4: Zaxira va tiklash

1. Sozlamalar → Zaxira nusxa
2. JSON fayl eksport qilinadi
3. Yangi telefon → Ilovani o'rnatadi
4. Sozlamalar → Tiklash → fayl tanlaydi
5. Barcha ma'lumotlar tiklandi

**Natija:** Ma'lumotlar xavfsiz ko'chirildi.

---

## Sayohat 5: Maqsadga erishish (Jasur)

1. Maqsad yaratadi: "Flutter o'rganish — 80%"
2. 5 ta bosqich qo'shadi
3. Har bosqichni bajarib progress yangilaydi
4. Dashboardda progress ko'rinadi
5. AI: "Maqsadingizning 60% bajarildi, davom eting!"

**Natija:** Uzoq muddatli maqsad kuzatildi.
